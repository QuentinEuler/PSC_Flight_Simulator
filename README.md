# PSC_Flight_Simulator

Dans ce dépôt, nous partageons notre travail effectué dans le cadre d'un projet scientifique de groupe. Son objectif est de prototyper une IA pour le playtest automatique de FlightSimulator. Nous avons principalement 2 approches : 

- une approche par apprentissage supervisée :
Avec cette technique, nous avons réussi à faire décoller et se stabiliser l'avion vericalement directement sur le jeux FlightSimulator.

- une approche par apprentissage par renforcement :
Cette approche propose 3 algorithmes : Q-Learning, Deep Q-Learning et Actor-Critic. Chacun de ces algorithmes ne fonctionne pas directement sur le jeu mais sur des simulateurs simplifiés à des fins de tests en amont d'une exportation vers FlightSimulator.
